LibNet

Machine learning tool for perfoming both inference and learning with probabilistic graphical models.

View project on GitHub

Libnet performs both learning and inference on graphical probabilistic models. The tool has been designed to run as fast as possible. As a result the tool has been written in the programming language C and a perfect hashing algorithm has been used in order to quickly look up the individual nodes. The main program gets compiled into a shared object (library of functions). There are two interfaces to the LibNet library: a command line interface and a R programming interface.

The program reads and writes data to flat files. The format of these flat files are specified in the documentation. With both the R and command line interface the user can specify both files and run time parameters.
